Custom Admin Menu is the best plugin for managing your WordPress Dashboard Menus.

== Description ==

While working with WordPress sites, you may feel the need for renaming or re-arranging WordPress menus. With this plugin, you can do it with just couple of clicks. You don't need to write a single line of code for changing menu names and re-arranging their position in the left menu panel.

You can also change color and lots of other stuffs in the paid version.

**Features**<br>
1. Change any menu name<br>
2. Re-arrange menus as you want<br>
3. Very easy user interface<br>
4. Beautiful Design<br>
5. Simple to use


== Installation ==

**Way 1:**<br>
1. Download the plugin file<br>
2. Unzip the file into a folder on your hard drive<br>
3. Upload via FTP into your plugins (/wp-content/plugins/) folder<br>
4. Activate the plugin from your dashboard<br>

**Way 2:**<br>
1. From Dashboard > Plugins > Add New > Upload the plugin<br>
2. Activate the plugin<br>
